# Greg Krieger - Google Search Simulation

A Google search simulation for **Greg Krieger** with interactive LinkedIn and Facebook profiles.

## Features

- **Google Search Results**: 5 pages of realistic search results for "Greg Krieger"
- **LinkedIn Profile**: Full interactive LinkedIn profile (click LinkedIn results to view)
  - Financial Analyst at Midwest Capital Advisors
  - University of Michigan education
  - Experience, skills, and activity posts
- **Facebook Profile**: Complete Facebook profile (click Facebook results to view)
  - Posts about sports, travel, and friends
  - Friends list and "People you may know"
  - About section with work and education

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173` to see the simulation.
